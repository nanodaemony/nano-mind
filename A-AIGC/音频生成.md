#### 提示词

1. 汉字（单字朗读）

语速最慢，让学生听清声母韵母和声调。

```json
{
    "voice": "<你的音色>",
    "text": "花",
    "model": "cosyvoice-v3-flash",
    "format": "mp3",
    "sampleRate": 22050,
    "volume": 50,
    "rate": 0.7,
    "pitch": 1.0,
    "seed": 42,
    "languageHints": ["zh"],
    "instruction": "请用严肃、清晰的教师语气朗读这个汉字，字正腔圆，发音饱满。"
}
```

2. 词汇（词语朗读）

语速适中，词内连贯但不过快。

```json
{
    "voice": "<你的音色>",
    "text": "花园",
    "model": "cosyvoice-v3-flash",
    "format": "mp3",
    "sampleRate": 22050,
    "volume": 50,
    "rate": 0.85,
    "pitch": 1.0,
    "seed": 42,
    "languageHints": ["zh"],
    "instruction": "请用严肃、清晰的教师语气朗读这个词语，注意词语内部的连贯性。"
}
```

3. 例句（句子朗读）

正常语速，自然流畅但仍保持教学感。

```json
{
    "voice": "<你的音色>",
    "text": "我家的后面有一个很大的花园。",
    "model": "cosyvoice-v3-flash",
    "format": "mp3",
    "sampleRate": 22050,
    "volume": 50,
    "rate": 1.0,
    "pitch": 1.0,
    "seed": 42,
    "languageHints": ["zh"],
    "instruction": "请用严肃、清晰的教师语气朗读这个句子，语速适中，断句自然，适合学生跟读。"      
}
```

多音字：
```
方式一：hotFix 参数（推荐）
   
通过拼音标注指定多音字的读音：

{
    "voice": "longshuo_v3",
    "text": "这个银行很长。",
    "model": "cosyvoice-v3-flash",
    "format": "mp3",
    "rate": 0.7,
    "pitch": 1.0,
    "seed": 42,
    "languageHints": ["zh"],
    "hotFix": {
        "pronunciation": [
            {"银行": "yin2 hang2"},
            {"很长": "hen3 chang2"}
        ]
    }
}

拼音格式是 <汉字>: "<拼音+数字声调>"，声调 1-4 分别对应一二三四声。也可以用 replace 直接替换文本： 

{
    "hotFix": {
        "replace": [
            {"行": "行走的行"}
        ]
    }
}
```
